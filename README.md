### Hexlet tests and linter status:
[![Actions Status](https://github.com/DSolokhin/frontend-project-12/actions/workflows/hexlet-check.yml/badge.svg)](https://github.com/DSolokhin/frontend-project-12/actions)

Учебный проект Hexlet: real-time чат с регистрацией, каналами, правами, веб-сокетами и хранением данных.
Фронтенд — React + Vite.
Бэкенд — @hexlet/chat-server.

Demo (Render)

Приложение развёрнуто на Render и доступно по ссылке:

https://chat-ds.onrender.com/

Локальный запуск приложения
1. Установить зависимости
make install

2. Собрать фронтенд
make build

3. Запустить приложение локально
make start


После запуска сервер будет доступен по адресу:  

http://localhost:5000
